# 迪尚 AI 数智化平台 UI 规范

## 一、核心设计理念

以「**功能聚焦 + 高效交互 + 行业适配**」为核心，参考扣子空间 “轻量化入口 + 模块化功能” 的布局逻辑，结合迪尚服装行业 AI 平台的业务属性（五大业务板块、多角色协同），打造 “易唤醒、快操作、强适配” 的 UI 体系，兼顾专业功能与易用性。

## 二、整体布局规范

### 2.1 布局结构（参考扣子空间 “导航 + 交互 + 输入” 三层架构）



| 区域     | 位置         | 核心功能                                                                 | 适配场景                      |
| ------ | ---------- | -------------------------------------------------------------------- | ------------------------- |
| 顶部状态栏  | 页面顶部（通栏）   | 1. 用户信息（角色标识、部门）；2. 系统通知（Agent 任务进度、审批提醒）；3. 功能入口（设置、帮助、退出）          | 全角色通用，快速查看关键信息            |
| 左侧板块导航 | 页面左侧（可折叠）  | 1. 五大业务板块入口（趋势与设计 / 生产与供应链 / 营销与销售 / 客户服务 / 内部管理与协同）；2. 常用 Agent 收藏夹 | 按角色权限动态显示可用板块             |
| 中间主交互区 | 页面中间（核心区域） | 模式 1：对话交互（Agent 聊天窗口，支持多轮沟通）；模式 2：任务看板（当前执行 / 待执行 / 已完成任务列表）         | 日常高频操作（对话唤醒 Agent）、复杂任务管理 |
| 右侧辅助区  | 页面右侧（可收起）  | 1. 任务进度详情；2. Agent 协作流程可视化；3. 关联系统数据预览（如 PLM 设计稿、SCM 库存数据）           | 复杂任务执行（如新款开发、团装定制）        |
| 底部输入区  | 页面底部（通栏）   | 1. Agent 唤醒输入框（支持 @Agent 名称、关键词检索）；2. 快捷功能按钮（发起任务、上传文件、切换模式）         | 快速发起需求、唤醒 Agent           |

### 2.2 布局尺寸规范



* 顶部状态栏高度：60px（固定，不随滚动变化）

* 左侧板块导航宽度：240px（折叠后 80px）

* 右侧辅助区宽度：320px（收起后 0px）

* 底部输入区高度：80px（固定，含输入框 + 快捷按钮）

* 中间主交互区：自适应剩余宽度（最小宽度 1000px，支持响应式缩放）

* 页面边距：全局统一 20px 内边距，模块间距 16px

## 四、组件规范

### 4.1 板块导航组件（左侧）



* **样式**：


  * 未选中态：文字 #86909C，背景透明，hover 时背景 #F2F3F5

  * 选中态：文字 #165DFF，左侧 3px 竖线（对应板块专属色），背景 #EEF4FF

  * 图标：搭配板块专属色，尺寸 24×24px，与文字间距 8px

* **交互**：


  * 点击切换板块，同步更新中间区 Agent 列表

  * 支持鼠标滚轮滚动（板块≥5 个时）

  * 顶部提供 “收藏夹” 入口（固定在前 3 位），显示用户常用 Agent

* **结构**：



```
板块导航项 = 板块图标 + 板块名称 + 未读提示（可选）

收藏夹 = 星星图标 + “常用Agent”文字 + 折叠/展开按钮
```

### 4.2 Agent 卡片组件（中间区 - 对话模式）



* **样式**：


  * 尺寸：宽度 100%，高度 80px，圆角 8px，边框 1px solid #E5E6EB

  * hover 态：边框色切换为对应板块专属色，阴影 0 2px 8px rgba (0,0,0,0.08)

  * 内容布局：左侧 Agent 图标（40×40px）+ 中间 Agent 名称 + 功能简介 + 右侧快捷按钮

* **信息层级**：


  * 主信息：Agent 名称（文字 #1D2129，字体 16px，加粗）

  * 次信息：核心功能简介（文字 #86909C，字体 14px，1 行省略）

  * 辅助信息：所属板块标签（对应板块专属色，圆角 4px，字体 12px）

* **快捷操作**：


  * 右侧默认显示 “唤醒” 按钮（文字 #165DFF，hover 时背景 #EEF4FF）

  * 常用 Agent 额外显示 “收藏” 按钮（空心星星，点击变实心 #165DFF）

### 4.3 对话交互组件（中间区 - 对话模式）



* **对话气泡**：


  * 用户气泡：右侧对齐，背景 #165DFF，文字白色，圆角 16px（右侧底部 0px）

  * Agent 气泡：左侧对齐，背景 #F2F3F5，文字 #1D2129，圆角 16px（左侧底部 0px）

  * 系统通知气泡：居中对齐，背景 #EEF4FF，文字 #165DFF，圆角 8px，字体 12px

* **交互元素**：


  * Agent 回复含 “结果文件”（如设计稿、报表）：显示文件预览卡片（宽度 200px，圆角 8px，hover 放大）

  * 多 Agent 协作：显示 “协作流程条”（横向，按执行顺序排列 Agent 图标，用箭头连接）

  * 多轮对话：支持 “追问” 按钮（文字 #165DFF，点击后输入框自动填充 “补充需求：”）

### 4.4 任务卡片组件（中间区 - 任务模式）



* **样式**：


  * 尺寸：宽度 100%，高度 100px，圆角 8px，边框 1px solid #E5E6EB，背景白色

  * 状态标识：左侧 4px 竖线（进行中 #165DFF / 完成 #00B42A / 延期 #F53F3F）

* **信息布局**：


  * 顶部：任务名称（文字 #1D2129，字体 16px）+ 所属板块标签（对应专属色）

  * 中部：任务描述（文字 #86909C，字体 14px，1 行省略）+ 进度条（高度 4px，背景 #F2F3F5）

  * 底部：创建时间（文字 #C9CDD4，字体 12px）+ 操作按钮（“查看详情”“取消任务”）

### 4.5 底部输入组件



* **输入框**：


  * 尺寸：宽度 100%，高度 48px，圆角 24px，边框 1px solid #E5E6EB， padding 0 16px

  * 聚焦态：边框色 #165DFF，阴影 0 0 0 2px rgba (22,93,255,0.1)

  * 提示文字：默认显示 “输入需求或 @Agent 唤醒（如：@款式创新 Agent 生成秋季西装设计稿）”（文字 #C9CDD4，字体 14px）

* **快捷功能按钮**：


  * 左侧：“切换模式”（对话 / 任务），图标尺寸 24×24px，颜色 #86909C，hover#165DFF

  * 中间：“上传文件”（支持设计稿、需求文档），图标尺寸 24×24px，颜色 #86909C，hover#165DFF

  * 右侧：“发送” 按钮，默认禁用（输入内容后激活），激活态背景 #165DFF，文字白色，圆角 24px

### 4.6 右侧辅助区组件



* **进度详情面板**：


  * 标题：文字 #1D2129，字体 16px，加粗，底部 border-bottom 1px solid #E5E6EB

  * 内容：分点显示 “当前执行 Agent”“已完成步骤”“剩余步骤”“预计完成时间”（文字 #86909C，字体 14px）

* **数据预览组件**：


  * 设计稿预览：支持 2D/3D 切换（3D 视图用 three.js 渲染，默认显示正面效果）

  * 报表预览：显示关键数据卡片（如成本金额、库存数量），数字 #1D2129，单位 #86909C

  * 系统对接入口：显示 “跳转 PLM/SCM/MES” 按钮（文字 #165DFF，hover 背景 #EEF4FF）

## 五、交互规范

### 5.1 Agent 唤醒交互



1. 输入框输入 “@” 触发 Agent 检索，下拉显示匹配的 Agent 列表（按板块分类，常用 Agent 置顶）

2. 选中 Agent 后，输入框自动补全 “@XXX Agent”，支持直接续写需求

3. 快捷唤醒：点击 Agent 卡片的 “唤醒” 按钮，输入框自动填充 “@XXX Agent ”，聚焦输入框

4. 多 Agent 协作：输入 “@通用 Agent 发起 XX 任务”，自动拆解任务并显示协作 Agent 列表，确认后启动流程

### 5.2 板块切换交互



1. 点击左侧板块导航，中间区 Agent 列表平滑切换（淡入淡出动画，时长 300ms）

2. 切换后自动定位到列表顶部，保留上次选中的 Agent 高亮状态

3. 折叠左侧导航后，鼠标 hover 图标显示板块名称浮窗（白色背景，阴影 0 2px 8px，时长 200ms）

### 5.3 任务操作交互



1. 发起任务：输入需求后点击 “发送”，弹出任务确认弹窗（显示任务名称、关联 Agent、预计耗时）

2. 任务跟踪：任务创建后自动跳转至任务模式，实时更新进度条和执行状态

3. 结果查看：任务完成后，Agent 气泡显示 “任务完成” 提示，右侧辅助区展示详细结果

4. 任务导出：支持导出任务报告（PDF/Excel 格式），导出按钮位于结果详情面板顶部

### 5.4 响应式适配



1. PC 端（≥1200px）：完整显示 “顶部 + 左侧 + 中间 + 右侧 + 底部” 五区域

2. 平板端（768px-1199px）：右侧辅助区默认收起，点击 “展开” 按钮弹出（覆盖中间区 30% 宽度）

3. 移动端（≤767px）：左侧板块导航改为底部 Tab 栏（五大板块图标 + 文字，高度 60px），右侧辅助区隐藏（需从任务详情页查看）

## 六、业务板块适配规范（核心差异化）

### 6.1 趋势与设计板块



* 专属元素：设计稿预览区支持 2D/3D 切换，3D 视图提供 “旋转”“缩放”“面料替换” 快捷按钮

* Agent 卡片额外显示 “设计风格标签”（如 “商务风”“国潮风”，对应颜色标签）

* 交互侧重：设计稿生成后支持 “一键同步 PLM”“局部改款” 快捷操作

### 6.2 生产与供应链板块



* 专属元素：排产进度条显示车间产能占比，库存预警用 “闪烁红点” 标识

* Agent 卡片额外显示 “数据来源标签”（如 “SCM 系统”“MES 系统”）

* 交互侧重：库存预测结果支持 “生成补货单”“调整安全阈值” 快捷操作

### 6.3 营销与销售板块



* 专属元素：营销素材预览支持 “多渠道适配”（电商详情页 / 海报 / 短视频脚本）切换

* Agent 卡片额外显示 “适用客群标签”（如 “商务客群”“年轻客群”）

* 交互侧重：客户画像生成后支持 “一键推送营销方案”“导出客户名单” 快捷操作

### 6.4 客户服务板块



* 专属元素：智能客服对话支持 “问题分类标签”（如 “订单查询”“售后反馈”）

* Agent 卡片额外显示 “响应时效”（如 “实时响应”“10 分钟内反馈”）

* 交互侧重：售后问题处理后支持 “满意度评价”（五星评分，提交后自动优化话术库）

### 6.5 内部管理与协同板块



* 专属元素：合同审查显示 “风险标记”（高风险 #F53F3F，中风险 #FF7D00，低风险 #00B42A）

* Agent 卡片额外显示 “权限等级”（如 “部门级”“公司级”）

* 交互侧重：培训 Agent 支持 “在线测试”“错题回顾”，报表 Agent 支持 “自定义维度导出”

## 七、字体与图标规范

### 7.1 字体规范



* 主字体：Inter（优先）、思源黑体、微软雅黑（ fallback 顺序）

* 字体大小：


  * 标题：18px（页面标题）、16px（模块标题、Agent 名称）

  * 正文：14px（需求输入、功能描述、对话内容）

  * 辅助文字：12px（时间戳、标签、提示说明）

* 字重：标题 600（加粗）、正文 400（常规）、强调文字 500（中等加粗）

### 7.2 图标规范



* 图标库：@arco-design/icons-vue + lucide-vue，补充服装行业专属图标（面料、模特、缝纫机等）

* 图标尺寸：导航图标 24×24px、按钮图标 20×20px、辅助图标 16×16px

* 图标颜色：默认 #86909C，选中态 #165DFF，板块专属图标用对应板块专属色
